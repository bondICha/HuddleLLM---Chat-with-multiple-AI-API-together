# Release Notes - KaTeX Math Rendering and Mermaid Diagram Support

## Version: v2.16.5
**Release Date:** 2026-05-01
**Base:** v2.16.4
**Highlights:** KaTeX math rendering, Mermaid diagram inline display, and system prompt diff modal.

---

## KaTeX Math Rendering

- Added `remark-math` + `rehype-katex` for rendering mathematical expressions
- Supports `$...$` for inline math and `$$...$$` for block equations
- Automatic normalization of LaTeX delimiters (`\(...\)` / `\[...\]`) used by GPT-style models
- System prompts updated with math notation guidance (all 4 languages)

---

## Mermaid Diagram Support

- AI-generated Mermaid code blocks render as inline diagrams (flowcharts, sequence diagrams, class diagrams, etc.)
- Preview/Code toggle to switch between rendered view and source code
- Expand modal for full-screen diagram viewing
- Copy button for copying Mermaid source code
- Always uses light theme for consistent appearance
- SVG sizing based on viewBox natural width (no unnecessary stretching)
- Max height limit (600px) with scroll for tall diagrams

---

## System Prompt Update Modal

- Added line-by-line diff comparison between current and new system prompts
- Language tab selector to preview prompts in all 4 languages (en, ja, zh-CN, zh-TW)
- Skip button to dismiss the update notification
