# Release Notes - Side Panel UI Improvements

## Version: v2.16.4
**Release Date:** 2026-04-30
**Base:** v2.16.3
**Highlights:** Side panel UI refinements, BotIcon adoption, and unused permission cleanup.

---

## Side Panel UI Improvements

- Bot avatar now uses `BotIcon` component (consistent with main app) instead of raw `<img>` tag
- Switched input mode from `compact` to `full` for better usability
- Buttons changed from `small` to `normal` size
- `onSubmit` now supports images, attachments, audio, video, and PDF files

---

## Permission Cleanup

- Removed `scripting` and `offscreen` permissions from manifest (unused)
- Removed offscreen PDF processor files (`pdf-processor.html`, `pdf-processor.ts`)

---

## Known Limitations

- Page content extraction (web tab context) is not yet supported due to host permission requirements
