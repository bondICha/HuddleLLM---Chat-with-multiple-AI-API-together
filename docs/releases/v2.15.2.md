# Release Notes - OpenAI Image Agent & Novita Model Expansion

## Version: v2.15.2
**Release Date:** 2026-03-11
**Base:** v2.15.1
**Highlights:** OpenAI Image (gpt-image-1.5) as Image Agent, new Novita models (GLM Image, Seedream 5.0 lite), and Quick Settings panel expanded to support image generation parameters.

---

## OpenAI Image Agent

- Added **gpt-image-1.5** as an Image Agent provider via the `openai-image` scheme.
- Supports both text-to-image and image editing (img2img) using the OpenAI Images API.
- Edit mode automatically activates when the user attaches images.

---

## Novita AI New Models

Two new models added to the Novita AI provider:

### GLM Image
- High-quality HD text-to-image generation with fine details and high consistency.
- Async API with selectable resolutions up to 2048×2048.
- Configurable watermark via Quick Settings.

### Seedream 5.0 lite
- Upgraded from Seedream 4.0 with higher resolution support (up to 3072×3072).
- Supports text-to-image and multi-image reference editing (up to 14 input images).
- Sequential image generation mode (`auto` / `disabled`).

---

## Quick Settings: Image Generation Parameters

The Quick Settings panel now exposes image tool parameters for Image Agent bots.

- **Enum parameters** (e.g., `size`, `quality`, `sequential_image_generation`) are shown as selectable buttons.
- **Boolean parameters** (e.g., `watermark_enabled`, `watermark`) are shown as `true` / `false` toggle buttons.
- A **`-` button** (not specified) is available for each parameter — selecting it lets the Agent decide the value freely without any override.

> All quick settings remain session-only and reset on reload.
