## Full Description（纯文本）

同时与多个 AI 聊天机器人对话的 Chrome 扩展程序。
向多个 AI 同时发送同一个问题，比较回答，快速找到最佳解答。

Bring Your Own API Key。本扩展程序需使用您自己的 API 密钥。
您可以逐一配置各家 API，也可以使用 OpenRouter，只需简单设置即可接入全球几乎所有的 AI。


◆ 使用场景 ◆

询问编程问题时，比较不同 AI 的回答，选出最简洁高效的代码。
寻求文案修改建议时，从多个视角获得更全面的反馈意见。
使用 /btw 命令，将多个 AI 的回答一并发送给另一个 AI，进行更深入的交叉比较分析。


◆ 主要功能 ◆

🌸 在同一界面同时与 ChatGPT、Gemini、Claude、GLM、MiMo 等多个 AI 对话并比较
🌺 原生 Web 搜索：OpenAI、Claude、Gemini 直接调用各自的官方搜索工具。其他模型回退至 HuddleLLM 内置的 Web 搜索（Function Call 方式）
🌻 多模态输入：图片、音频、视频与 PDF
🌼 Chrome 侧边栏：无需离开当前标签页，即可快速打开单个 Bot 对话
🪷 Omnibox 集成：在地址栏输入"hl 关键词"即可瞬间启动
🏵️ 语音转文字：通过 OpenAI Whisper 或 Gemini 将音频转为文字后发送
💐 自定义与社区提示词库
💮 会话历史本地保存，支持完整恢复与导入 / 导出
🌹 Image Agent：用自然语言描述想要的图片，由 LLM 生成优化后的提示词再调用图像生成 API（OpenAI Image、Gemini、Chutes AI、Novita AI、Replicate）
🌷 Quick Settings 面板：按会话快速调整思考等级与图像生成参数
🪻 /btw 命令：将多个 AI 的回答一并发送给另一个 AI，在独立弹窗中进行交叉比较与深度分析


◆ 支持的服务商 ◆

原生 API：OpenAI（Chat Completions / Responses API）、Google Gemini、Anthropic Claude

OpenAI 兼容：OpenRouter、DeepSeek、Qwen、xAI Grok、Z.AI GLM、Together AI、Fireworks AI、Hyperbolic、DeepInfra、Nebius 及任意自定义端点


◆ 隐私说明 ◆

所有数据（API 密钥、会话历史）仅保存在浏览器本地。无遥测、无需账号、不向 HuddleLLM 服务器发送任何数据。
