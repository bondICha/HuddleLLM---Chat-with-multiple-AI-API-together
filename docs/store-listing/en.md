## Full Description

Engage in conversations with multiple AI chatbots simultaneously.

HuddleLLM allows you to chat with multiple AI chatbots at once, making it easy to compare their responses and get the best answer for any task.

**Benefits**

When asking about coding, you can compare responses from different chatbots and choose the cleanest, most efficient code.
When seeking advice on text revision, you can receive more comprehensive feedback from multiple perspectives.
With the /btw command, you can even send all AI responses to another AI for cross-comparison and deeper analysis.

**Key Features**

- Chat with ChatGPT, Gemini, Claude, and more side by side in a single screen
- Native Web Search: OpenAI Responses, Claude, and Gemini use their own search tools for accurate, low-overhead results. Other providers fall back to HuddleLLM's built-in web search via Function Calls.
- Multimodal input: images, audio, video, and PDFs
- Image Agent: describe what you want in natural language and let an LLM generate an optimized prompt before sending it to an image generation API (supports OpenAI Image, Gemini, Chutes AI, Novita AI, Replicate)
- Quick Settings panel: adjust reasoning level and image-generation parameters per session
- Chrome Side Panel: open individual bots for quick chats without leaving your current tab
- Omnibox integration: type `hl <keyword>` in the address bar to launch HuddleLLM instantly
- Speech-to-Text: transcribe audio via OpenAI Whisper or Gemini and send it as text
- Prompt Library with custom and community prompts
- Local conversation history with full export / import

**Supported Providers**

Native APIs: OpenAI (Chat Completions / Responses API), Google Gemini, Anthropic Claude

OpenAI-compatible: OpenRouter, DeepSeek, Qwen, xAI Grok, Z.AI GLM, Together AI, Fireworks AI, Hyperbolic, DeepInfra, Nebius, and any custom endpoint

**Privacy**

HuddleLLM stores all data (API keys, conversation history) locally in your browser only. No telemetry, no account required, no data sent to HuddleLLM servers.
