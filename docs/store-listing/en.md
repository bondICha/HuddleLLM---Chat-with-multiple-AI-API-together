## Full Description (plain text)

A Chrome extension that lets you chat with multiple AI chatbots at the same time.
Send the same question to multiple AIs simultaneously, compare their responses, and find the best answer fast.

Bring Your Own API Key. This extension works with your own API credentials.
You can configure each provider directly, or use OpenRouter for a single setup that covers virtually every AI in the world.


◆ BENEFITS ◆

When asking about coding, compare responses from different AIs and pick the cleanest, most efficient code.
When seeking writing feedback, get more comprehensive suggestions from multiple perspectives at once.
Use the /btw command to send all AI responses to another AI for cross-comparison and deeper analysis.


◆ KEY FEATURES ◆

🌸 Chat and compare multiple AIs (ChatGPT, Gemini, Claude, GLM, MiMo, and more) side by side in one screen
🌺 Native Web Search: OpenAI, Claude, and Gemini use their own official search tools directly. Other models fall back to HuddleLLM's built-in web search via Function Calls.
🌻 Multimodal input: images, audio, video, and PDFs
🌼 Chrome Side Panel: open individual bots for quick chats without leaving your current tab
🪷 Omnibox integration: type "hl keyword" in the address bar to launch HuddleLLM instantly
🏵️ Speech-to-Text: transcribe audio via OpenAI Whisper or Gemini and send it as text
💐 Custom and community Prompt Library
💮 Local conversation history with full session restore and export / import
🌹 Image Agent: describe what you want in natural language — an LLM generates an optimized prompt and sends it to an image generation API (OpenAI Image, Gemini, Chutes AI, Novita AI, Replicate)
🌷 Quick Settings panel: adjust reasoning level and image-generation parameters per session
🪻 /btw command: send multiple AI responses to another AI for cross-comparison and deep analysis in a standalone popup


◆ SUPPORTED PROVIDERS ◆

Native APIs: OpenAI (Chat Completions / Responses API), Google Gemini, Anthropic Claude

OpenAI-compatible: OpenRouter, DeepSeek, Qwen, xAI Grok, Z.AI GLM, Together AI, Fireworks AI, Hyperbolic, DeepInfra, Nebius, and any custom endpoint


◆ PRIVACY ◆

All data (API keys, conversation history) is stored locally in your browser only. No telemetry, no account required, no data sent to HuddleLLM servers.
