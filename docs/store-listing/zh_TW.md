## Full Description（純文字）

同時與多個 AI 聊天機器人對話的 Chrome 擴充功能。
向多個 AI 同時發送同一個問題，比較回答，快速找到最佳解答。

Bring Your Own API Key。本擴充功能需使用您自己的 API 金鑰。
您可以逐一設定各家 API，也可以使用 OpenRouter，只需簡單設定即可接入全球幾乎所有的 AI。


◆ 使用情境 ◆

詢問程式設計問題時，比較不同 AI 的回答，選出最簡潔高效的程式碼。
尋求文案修改建議時，從多個角度獲得更全面的意見回饋。
使用 /btw 指令，將多個 AI 的回答一併發送給另一個 AI，進行更深入的交叉比較分析。


◆ 主要功能 ◆

🌸 在同一畫面同時與 ChatGPT、Gemini、Claude、GLM、MiMo 等多個 AI 對話並比較
🌺 原生 Web 搜尋：OpenAI、Claude、Gemini 直接呼叫各自的官方搜尋工具。其他模型回退至 HuddleLLM 內建的 Web 搜尋（Function Call 方式）
🌻 多模態輸入：圖片、音訊、影片與 PDF
🌼 Chrome 側邊欄：無需離開目前分頁，即可快速開啟單一 Bot 對話
🪷 Omnibox 整合：在網址列輸入「hl 關鍵字」即可瞬間啟動
🏵️ 語音轉文字：透過 OpenAI Whisper 或 Gemini 將音訊轉為文字後送出
💐 自訂與社群提示詞庫
💮 對話歷史本機保存，支援完整還原與匯入 / 匯出
🌹 Image Agent：用自然語言描述想要的圖片，由 LLM 產生最佳化的提示詞再呼叫圖像生成 API（OpenAI Image、Gemini、Chutes AI、Novita AI、Replicate）
🌷 Quick Settings 面板：依工作階段快速調整思考等級與圖像生成參數
🪻 /btw 指令：將多個 AI 的回答一併發送給另一個 AI，在獨立彈出視窗中進行交叉比較與深度分析


◆ 支援的服務商 ◆

原生 API：OpenAI（Chat Completions / Responses API）、Google Gemini、Anthropic Claude

OpenAI 相容：OpenRouter、DeepSeek、Qwen、xAI Grok、Z.AI GLM、Together AI、Fireworks AI、Hyperbolic、DeepInfra、Nebius 及任意自訂端點


◆ 隱私說明 ◆

所有資料（API 金鑰、對話歷史）僅保存在瀏覽器本機。無遙測、無需帳號、不向 HuddleLLM 伺服器傳送任何資料。
